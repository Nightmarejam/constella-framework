# Frameworks — versioned snapshots

This directory holds **frozen, versioned snapshots** of the Constella framework
(e.g. `constella_1.6.md`). Nothing here is edited after the fact — new versions get
new files. The **living documents** are in `docs/constella/`; when the two disagree,
the living docs win and a snapshot records what a version *said*, not what is true now.
