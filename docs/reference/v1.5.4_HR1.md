## Math & Mechanics

### 1) Astris (A) — Accrual & Decay

- **Accrual per task:** `A_gain = B * C_q * V_p * R_s`
  - `B` = base bounty for task
  - `C_q` = contribution quality (0.6–1.4 via rubric)
  - `V_p` = verified proof factor (0.8–1.2; repo links, QR/NFC logs)
  - `R_s` = redundancy score (1.0 if unique; 1.1 if verified backup role)

- **Activity decay (weekly, applied on Sundays 00:00 UTC):**
  - If active (≥1 verified action/week): **no decay**.
  - If inactive `w` weeks: `A_new = A_prev * (1 - d)^w`, with `d = 0.02` (2%/week).
  - **Plateau protection:** contributions flagged “foundational” (>95th percentile impact) are capped at **max 10% total decay** lifetime.

**Example:** A=1000, 3 inactive weeks → `1000*(0.98^3)=941.192`.

### 2) Auctor (Au) — Equality Vote with Light Decay

- **Base grant:** 1 Au per verified person (proof-of-personhood).
- **Inactivity decay:** if no governance participation for `q` quarters:
  `Au_new = Au_prev * (1 - 0.05)^q` (5%/quarter), floor = 0.5.
- **Reactivation:** any two actions in a quarter (vote, review, meeting) restore to pre‑decay level up to 1.0.

### 3) Cross‑Vault Rebalancing

- **Signal:** vault coverage ratio `CR = Reserves / (3*AvgMonthlyOutflow)`
- **Action:** if `CR < 0.8` for 2 consecutive weeks, auto‑proposal to move up to **min(5%, gap)** from vaults with `CR > 1.2` (subject to Auctor vote + Astris weighted advisory).
- **Guardrail:** single move cannot drop any donor vault below `CR=1.0`.

### 4) Civic Floor (UCF) Allocation

- **Sources:** (a) Vault surpluses ≥ CR1.5, (b) OPP (Open Palm) pledges, (c) Genesis interest, (d) partner grants.
- **Monthly allocation:** `UCF = min( BasePop*H , 0.25*(Surplus_total) )`
  - `H` = per‑capita floor (pilot default: **$200** equiv/mo in goods/services)
- **Priority:** food, shelter, utilities, transit, basic care; **in‑kind first**, cash last.

## Verification, Privacy & Anti‑Gaming

- **Local-first ledger:** QR/NFC app stores signed events on device; syncs hashed proofs to network.
- **Zero-Knowledge Badge (ZKB):** venue verifies “Member-in-good-standing” without seeing PII; rotating keys every 30 days.
- **Anomaly detection:** heuristics on bursty contributions, device reuse, geo‑impossible overlaps; human steward review escalations.
- **Dual attestation:** risky tasks (PSL2–3) require **two independent verifications** (device + peer; or device + photo+EXIF redacted).

## Governance Edge Cases

- **Deadlock resolution:** if two rounds end tied → invoke **Randomized Civic Jury (12)** from active Auctor pool; simple majority; one‑time binding.
- **Facilitator load caps:** max 5 active cases/facilitator; queue auto‑rotation; cooldown 2 weeks after heavy case.
- **Appeals:** single appeal to **Meta‑Steward Panel** (3 rotating seniors) within 7 days; decisions published with anonymized rationale.

## Project Safety Levels (PSL)

- **PSL0:** documentation, discussion — standard review.
- **PSL1:** software prototypes — code review + sandboxing.
- **PSL2:** light hardware / field — partner org required; safety checklist.
- **PSL3:** bio/infra/risk — licensed partner + IRB/ethics approval + contingency plan + insurance proof.

## RME Integration ([RME-FOCUS])

- **Placement:** after Civic Orchestration, before Vault Funding.
- **Weights:** default priority score `P = 0.5*Impact + 0.3*Feasibility + 0.2*Readiness` (0–1 each; reviewers provide).
- **Partner MoU bullets:** scope, data sharing, IP stance (default: **community‑licensed results, partner retains background IP**), safety level, exit clause.
- **Outputs:** ranked list, partner list (primary/alternates), bottleneck log, next‑step plan (2–4 weeks).

## Pilot: Eden Housing (12 Weeks)

- **Cohort:** 30–40 residents; 1 building.
- **Threads:** maintenance prioritization, energy retrofit scoping, shared space norms.
- **Tech:** QR/NFC check‑ins; weekly assemblies; RME for any researchy thread.
- **Metrics:** participation ≥65%; decision time ≤7 days; fulfillment ≥80%; conflict resolution avg ≤10 days; UCF utilization & satisfaction.
- **Stop conditions:** repeated CR<0.6, safety incident > PSL1 without partner, participation <30% for 3 weeks.

## Updated Experimental Stubs (split & prioritized)

### Stub A — Atom‑as‑Planet Spectral Ratios 🧭  (**Feasibility: Medium; Impact: High**)

- **Hypothesis:** spectral line intervals exhibit harmonic patterns analogous to planetary ratios (after normalization).
- **Method:** high‑resolution spectroscopy (H/He); Fourier analysis of line spacings; compare to Keplerian normalized sets.
- **Controls:** temp/pressure/EM shielding; instrument calibration.
- **Success:** statistically significant recurring ratios across elements or states (p<0.01, effect size ≥0.5).  
- **Partners:** NIST, CERN, MIT CUA.  
- **Implementation Consideration:** instrument precision & beamtime access.

### Stub B — Body–Cosmos Proportional Mapping 🧭✨  (**Feasibility: High; Impact: Medium**)

- **Hypothesis:** subsets of human skeletal ratios align with scaled orbital distances within tolerance.
- **Method:** n≥500 anthropometry; preregistered analysis; Bonferroni‑corrected tests; report negatives.
- **Success:** pre‑declared ratio set passes FDR<0.1 with cross‑validation.  
- **Partners:** Mayo Clinic, CfA, ESA.  
- **Implementation Consideration:** measurement error & sampling bias.

### Stub C — Breath–Tidal Resonance 🧭  (**Feasibility: High; Impact: Medium**)

- **Hypothesis:** population‑level respiration variability correlates with local tidal cycles beyond circadian effects.
- **Method:** wearable RR over 30 days; mixed models with tide data; control for activity & time.  
- **Success:** consistent cross‑site correlation with preregistered effect direction.  
- **Partners:** WHOI, Stanford Sleep, Scripps.  
- **Implementation Consideration:** coastal site logistics.
