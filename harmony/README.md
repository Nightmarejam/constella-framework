## Three Modules

2. **Feet Module** - Second-stage gain: grounding, propulsion, windlass mechanism
3. **Hands Module** - Micro-tuning: precision adjustment, lateral interface

- **Phase Flip**: Biomechanical transition where anterior-descending fascial tension exchanges with posterior-ascending stabilization
